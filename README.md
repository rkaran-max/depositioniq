# depositioniq
LLM-powered legal reasoning agent for deposition analysis and contradiction detection

## Project Goal
This project explores how agent systems can automate parts of legal reasoning workflows that traditionally require significant human review.

The system will:
- Ingest deposition transcripts
- Segment testimony into structured chunks
- Extract key witness claims
- Identify contradictions across testimony
- Verify contradictions to reduce false positives
- Generate suggested cross-examination questions
- Provide interpretable reasoning traces with source citations

## Planned Architecture
1. Transcript ingestion + preprocessing
2. Structured claim extraction
3. Contradiction detection
4. Verification layer
5. Reasoning / cross-exam generation
6. UI layer
